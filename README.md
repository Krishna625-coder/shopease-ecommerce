# E-Commerce Website

A modern and responsive e-commerce website built using React. The project provides a clean shopping experience with product listings, category filtering, search functionality, and a user-friendly interface.


## Features

- Responsive design
- Product listing page
- Product search functionality
- Category filtering
- Shopping cart UI
- Modern user interface
- Fast performance

## Tech Stack

- React.js
- JavaScript 
- HTML5
- CSS3
- Vite

## Installation

```bash
git clone https://github.com/Krishna625-coder/ecommerce-website.git
cd ecommerce-website
npm install
npm run dev
```

## Project Structure

```text
src/
```
